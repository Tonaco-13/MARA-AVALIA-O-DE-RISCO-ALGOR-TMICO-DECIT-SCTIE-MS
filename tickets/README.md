## Como usar com o Claude Code

Abra o terminal no diretório MARA e instrua:

```
Leia o ticket em tickets/01-limpeza-dependencias.md e execute-o.
```

Ou para executar em sequência:

```
Leia os tickets em tickets/ na ordem numérica e execute um por vez,
fazendo commit após cada ticket concluído.
```
